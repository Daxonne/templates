# Daxonne Templates

> Official template registry for [Daxonne](https://github.com/Daxonne/core) — language-agnostic code generation from your database schema.

[![Templates](https://img.shields.io/badge/templates-4-blue)](registry.json)
[![License](https://img.shields.io/github/license/Daxonne/templates)](LICENSE)

## Available templates

| Name | Language | Description |
|---|---|---|
| [`csharp-dapper`](#csharp-dapper) | C# | Records + Dapper CRUD repositories |
| [`typescript-prisma`](#typescript-prisma) | TypeScript | Interfaces + Prisma client services |
| [`java-jpa`](#java-jpa) | Java | Jakarta EE entities + Spring Data JPA repositories |
| [`python-sqlalchemy`](#python-sqlalchemy) | Python | SQLAlchemy models + async repositories |

Install any template with:

```bash
daxonne add <name>
```

## Template details

### `csharp-dapper`

Generates C# code for .NET projects using [Dapper](https://github.com/DapperLib/Dapper) as the micro-ORM.

**Generated files (per table):**

| File | Description |
|---|---|
| `{Table}Dto.cs` | Immutable `record` with one property per column |
| `{Table}Repository.cs` | Dapper repository with `GetAllAsync`, `GetByIdAsync`, `InsertAsync`, `DeleteAsync` |

**Example output — `CustomerDto.cs`:**
```csharp
// Auto-generated by Daxonne — do not edit manually

public record CustomerDto(
    long Id,
    string FirstName,
    string LastName,
    string? Email,
    decimal Balance,
    DateOnly JoinedAt
);
```

**Example output — `CustomerRepository.cs`:**
```csharp
// Auto-generated by Daxonne — do not edit manually

using System.Data;
using Dapper;

public class CustomerRepository
{
    private readonly IDbConnection _db;
    public CustomerRepository(IDbConnection db) => _db = db;

    public async Task<IEnumerable<CustomerDto>> GetAllAsync() { ... }
    public async Task<CustomerDto?> GetByIdAsync(long id) { ... }
    public async Task<int> InsertAsync(CustomerDto dto) { ... }
    public async Task<int> DeleteAsync(long id) { ... }
}
```

### `typescript-prisma`

Generates TypeScript code for projects using [Prisma](https://www.prisma.io/) as the ORM.

**Generated files (per table):**

| File | Description |
|---|---|
| `{Table}.ts` | TypeScript `interface` with correctly typed optional properties |
| `{Table}Service.ts` | Prisma client service with `getAll`, `getById`, `create`, `delete` |

**Example output — `Customer.ts`:**
```typescript
// Auto-generated by Daxonne — do not edit manually

export interface Customer {
  id: number;
  firstName: string;
  lastName: string;
  email?: string;
  balance: number;
  joinedAt: Date;
}
```

### `java-jpa`

Generates Java code for Spring Boot projects using [Spring Data JPA](https://spring.io/projects/spring-data-jpa) and the Jakarta EE persistence API.

**Generated files (per table):**

| File | Description |
|---|---|
| `{Table}.java` | `@Entity` class with JPA annotations and getters/setters |
| `{Table}Repository.java` | Spring Data `JpaRepository` interface |

**Example output — `Customer.java`:**
```java
// Auto-generated by Daxonne — do not edit manually

import jakarta.persistence.*;

@Entity
@Table(name = "CUSTOMER")
public class Customer implements Serializable {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    @Column(name = "ID", nullable = false)
    private Long id;

    @Column(name = "FIRST_NAME", nullable = false)
    private String firstName;

    // ...getters and setters...
}
```

### `python-sqlalchemy`

Generates Python code for projects using [SQLAlchemy](https://www.sqlalchemy.org/) with the async session interface.

**Generated files (per table):**

| File | Description |
|---|---|
| `{table}.py` | SQLAlchemy declarative model with column definitions |
| `{table}_repository.py` | Async repository with `get_all`, `get_by_id`, `insert`, `delete` |

**Example output — `customer.py`:**
```python
# Auto-generated by Daxonne — do not edit manually

from sqlalchemy import Column, BigInteger, String, Numeric, Date
from sqlalchemy.orm import DeclarativeBase

class Base(DeclarativeBase):
    pass

class Customer(Base):
    __tablename__ = "CUSTOMER"

    id = Column(BigInteger, nullable=False, primary_key=True)
    first_name = Column(String, nullable=False)
    last_name = Column(String, nullable=False)
    email = Column(String)
    balance = Column(Numeric)
    joined_at = Column(Date, nullable=False)
```

## Template format

Each template directory contains:

```
<name>/
├── template.json       ← manifest
└── *.hbs               ← Handlebars template files
```

**`template.json`:**
```json
{
  "name": "my-template",
  "version": "1.0.0",
  "language": "csharp",
  "description": "Short description",
  "author": "your-handle",
  "files": [
    { "template": "dto.hbs",  "output": "{{PascalCase name}}Dto.cs",  "per": "table" },
    { "template": "index.hbs", "output": "index.ts", "per": "schema" }
  ]
}
```

**`per`** can be:
- `"table"` — one output file per table
- `"schema"` — one file for the whole schema

### Available Handlebars helpers

| Helper | Example input | Output |
|---|---|---|
| `{{PascalCase name}}` | `USER_ACCOUNT` | `UserAccount` |
| `{{CamelCase name}}` | `USER_ACCOUNT` | `userAccount` |
| `{{SnakeCase name}}` | `USER_ACCOUNT` | `user_account` |
| `{{CSharpType type}}` | `decimal` | `decimal` |
| `{{TypeScriptType type}}` | `long` | `number` |
| `{{JavaType type}}` | `decimal` | `BigDecimal` |
| `{{PythonType type}}` | `datetime` | `datetime` |
| `{{SQLAlchemyType type}}` | `long` | `BigInteger` |
| `{{JoinColumns columns}}` | — | `ID, FIRST_NAME, EMAIL` |
| `{{JoinParams columns}}` | — | `:Id, :FirstName, :Email` |
| `{{PrimaryKeyColumn columns}}` | — | `ID` |
| `{{PrimaryKeyCamelCase columns}}` | — | `id` |
| `{{PrimaryKeyType columns}}` | — | `long` (C#) |
| `{{TypeScriptPrimaryKeyType columns}}` | — | `number` |
| `{{JavaPrimaryKeyType columns}}` | — | `Long` |
| `{{PythonPrimaryKeyType columns}}` | — | `int` |

### Template data shape

```json
{
  "name": "TABLE_NAME",
  "columns": [
    { "name": "ID",    "type": "long",   "nullable": false, "isPrimary": true  },
    { "name": "EMAIL", "type": "string", "nullable": true,  "isPrimary": false }
  ],
  "primaryKeys": ["ID"],
  "foreignKeys": [
    { "column": "DEPT_ID", "referencedTable": "DEPARTMENTS", "referencedColumn": "ID" }
  ]
}
```

### Internal type mapping

| InternalType | Oracle | PostgreSQL | MySQL | C# | TypeScript | Java | Python |
|---|---|---|---|---|---|---|---|
| `string` | VARCHAR2, CHAR, CLOB | varchar, text, char | VARCHAR, TEXT, CHAR | `string` | `string` | `String` | `str` |
| `int` | NUMBER(≤9) | integer, smallint | INT, SMALLINT, TINYINT | `int` | `number` | `Integer` | `int` |
| `long` | NUMBER(10–18) | bigint | BIGINT | `long` | `number` | `Long` | `int` |
| `decimal` | NUMBER(p,s>0), FLOAT | numeric, real, double | DECIMAL, FLOAT, DOUBLE | `decimal` | `number` | `BigDecimal` | `Decimal` |
| `bool` | — | boolean | TINYINT(1) | `bool` | `boolean` | `Boolean` | `bool` |
| `date` | DATE | date | DATE | `DateOnly` | `Date` | `LocalDate` | `date` |
| `datetime` | TIMESTAMP | timestamp | DATETIME, TIMESTAMP | `DateTime` | `Date` | `LocalDateTime` | `datetime` |
| `bytes` | BLOB, RAW | bytea | BLOB, BINARY | `byte[]` | `Uint8Array` | `byte[]` | `bytes` |
| `uuid` | — | uuid | — | `Guid` | `string` | `UUID` | `UUID` |
